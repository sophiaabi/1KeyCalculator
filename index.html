App Instructions

This application's default is a 5-Key interface. However, it can switch back and forth between a 5-Key interface and a 1-Key interface.

  • To navigate the 5-Key interface, press the right arrow key to move forward, left arrow key to move back, up arrow key to move up, and down arrow key to move down. Then to select a button, press the Shift key.
  • To navigate they 1-Key interface, press Shift to move forward from left to right through the buttons. Then to select a button, hold down the Shift key.
  • To switch between interfaces, navigate to the top buttons and select the button that corresponds to whatever interface you would like to use.

Test Results

After conducting the tests, it is evident that 5 key input mode performed better. It was very easy to make errors with the 1 key input, since you are pressing 1 key the whole time, so it was easy to accidentally skip over the intended button if you clicked too fast, resulting in needed to go through all the buttons again.

  • 1 Key Input- Average time to execute: 1 minute 52 seconds
    • Trial 1: 2 minutes 1 second - 3 Errors
    • Trial 2: 1 minute 46 seconds - 2 Errors
    • Trial 3: 1 minute 51 seconds - 3 errors
  • 5- Key Input- Average time to execute: 1 minute 12 seconds
    • Trial 1: 1 min 18 seconds - 1 Error
    • Trial 2: 1 minute 11 second- 1 Error
    • Trial 3: 1 minute 7 seconds - 2 Errors
